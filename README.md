# AI-Powered Restaurant Customer Support Agent (Telegram)

This project is a conversational AI agent built for a restaurant to automate customer support and order management via Telegram.

## Features
- Menu browsing with real-time availability
- Order placement with confirmation flow
- Automated bill calculation
- Order storage using Google Sheets
- n8n-based workflow orchestration
- Telegram Bot integration

## Tech Stack
- Telegram Bot API
- n8n (Workflow Orchestration)
- LLM (Groq / OpenAI)
- Google Sheets (Data Storage)
