# AI-Powered Restaurant Customer Support Agent (Telegram)

This project is a conversational AI agent built for a restaurant to automate
customer support and food ordering via Telegram.

## Features
- Menu browsing with real-time availability
- Order placement with confirmation flow
- Automated bill calculation
- Order storage using Google Sheets
- Workflow orchestration using n8n

## Tech Stack
- Telegram Bot API
- n8n (Workflow Automation)
- LLM ( Gemini )
- Google Sheets

## Architecture
The system uses Telegram as the interface, n8n as the orchestration layer,
and Google Sheets as a lightweight database.

## Demo
- Telegram chat screenshots available in `/screenshots`
- n8n workflow export available in `/workflows`

## Security
- No API keys committed
- Webhooks protected
